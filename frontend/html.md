###Inline styles
```
<p style="font-size:10px; color:green;"> 
Some text for you to make tiny!
</p>
```
###A table example
```
<table border="1px">
	<thead>
		<tr>
			<th colspan=2>Famous Monster by birth year</th>
		</tr>
		<tr>
			<th>Famous Monster</th>
			<th>Birth Year</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>King Kong</td>
			<td>1933</td>     
		</tr>
		<tr>
			<td>Dracula</td>
				<td>1897</td>
		</tr>
	</tbody>
</table>
```
